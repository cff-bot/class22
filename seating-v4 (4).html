📖 사용법

① 교실 설정

  • ⚙️ 교실 설정에서 분단 수·줄 수 입력 후 교실 생성
  • 기본 3분단×4줄 = 24자리(12쌍)

② 자리 설정 (성별 지정)

  • 🎨 자리 설정 버튼 켜기
  • 각 자리 클릭 → 남↔여↔미설정 토글
  • 홀수(25명 등): 3인 짝 필요 → 옆 자리에 같은 성별 하나 더 지정
  • 완료 후 버튼 다시 눌러 종료·저장

③ 학생 등록

  • 왼쪽에서 직접 입력 또는 👥 이름 업로드
  • 파일 형식: 한 줄에 "홍길동 남" 또는 "홍길동,여"

④ 추첨

  • 🎲 추첨하기 → 지정된 성별에 맞게 자동 배정
  • 히스토리 누적 → 이전 짝꿍 자동 회피
  • 모두 만났으면 가장 오래 전 만난 짝부터 재배정
  • 직전 짝꿍 재배정 시 ⚠ 재회 알림

⑤ 저장

  • 추첨 후 ✦ 결과 저장으로 히스토리 기록
  • 💾 저장: 브라우저 로컬 저장 / 📤 내보내기: 파일 백업
