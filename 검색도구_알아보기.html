인터넷에서 원하는 정보를 똑똑하게 찾는 방법

🌐 어떤 검색 도구가 있을까요?
🤔 검색 도구란?

검색 도구는 사용자가 원하는 정보를 찾을 수 있도록 도와주는 프로그램이에요. 검색어를 입력하면 관련 정보를 찾아서 보여줘요.

🔵
구글 (Google)
전 세계에서 가장 많이 사용하는 검색 도구예요. 다양한 언어와 나라의 자료를 찾을 수 있어요.
세계 1위
🟢
네이버 (Naver)
우리나라에서 가장 많이 사용하는 검색 도구예요. 한국어 자료가 많고 뉴스, 블로그, 카페 등을 함께 볼 수 있어요.
국내 1위
🔴
유튜브 (YouTube)
동영상 전문 검색 도구예요. 직업 소개 영상, 어떤 일을 하는지 보여주는 영상을 찾을 때 좋아요.
동영상 검색
🤖
인공지능 검색
클로드, 챗GPT 같은 인공지능에게 질문하면 답변을 정리해줘요. 단, 틀린 내용이 있을 수 있어서 확인이 필요해요.
AI 도우미
📋 검색 결과의 종류

검색 도구에서는 한 가지가 아니라 여러 종류의 자료를 볼 수 있어요!

  • 📰 뉴스 — 관련 사건이나 새로운 소식을 알려줘요
  • 🖼️ 이미지 — 그림이나 사진이 필요할 때, 출처를 꼭 밝혀야 해요
  • 📖 사전 — 낱말의 뜻과 관련 여러 내용을 알 수 있어요
  • ✍️ 블로그 — 관심 있는 사람들이 정리한 자료를 볼 수 있어요
  • 🎬 동영상 — 살아 움직이는 영상으로 자세히 볼 수 있어요
  • 📚 도서 — 관련 책 정보를 찾을 수 있어요

🌟 좋은 검색어 만드는 공식

[직업 이름] + [알고 싶은 것]
예) "작곡가 하는 일" / "원예사 되는 방법" / "드론 조종사 필요한 자격증"

⚠️ 정보를 찾을 때 주의할 점
🧐
모든 정보가 맞는 건 아니에요!
인터넷에는 누구나 글을 올릴 수 있어서 틀린 내용이 있을 수 있어요. 여러 곳에서 같은 내용이 나오는지 확인해요.
🤖
인공지능도 틀릴 수 있어요!
챗GPT, 클로드 같은 인공지능은 답변을 잘 정리해주지만, 가끔 틀린 정보를 자신 있게 말하기도 해요. 중요한 내용은 꼭 다른 곳에서도 확인해요.
🖼️
이미지는 출처를 밝혀요!
인터넷에서 찾은 그림이나 사진을 사용할 때는 어디서 가져왔는지 출처를 꼭 적어야 해요.
🔗
광고와 정보를 구별해요!
검색 결과 맨 위에 나오는 것이 항상 가장 좋은 정보는 아니에요. '광고'라고 적혀 있는 것은 돈을 내고 올린 글이에요.
🔒
개인 정보는 입력하지 않아요!
검색할 때 내 이름, 학교, 전화번호 같은 개인 정보는 절대 입력하지 않아요.
