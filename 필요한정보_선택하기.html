📋 정보를 선택하는 세 가지 기준
검색을 하면 수많은 자료가 나와요. 하지만 그 자료가 모두 좋은 자료는 아니에요.
아래 세 가지 기준으로 꼼꼼하게 따져보고 자료를 선택해야 해요!
1
🎯 목적에 적합한 정보인가?
내가 찾으려는 목적에 맞는지를 따져 보는 기준이에요. 선택한 정보가 목적에 알맞은지 판단해야 해요.
✓
주제와 관련 있는 정보인가?
✓
너무 어렵거나 쉬운 내용은 아닌가?
✓
내가 알고 싶은 내용을 담고 있는가?
💬 예를 들어 "드론 조종사 하는 일"을 알고 싶은데 드론 장비를 파는 쇼핑 사이트가 나왔다면? → 목적에 맞지 않아요!
2
📅 최신 내용을 반영한 정보인가?
새로운 정보인지를 따져 보는 기준이에요. 오래된 정보 가운데 일부는 내용이 바뀌어서 잘못된 정보일 수도 있어요.
✓
최신 정보를 사용했는가?
✓
찾으려는 내용은 최신 정보가 꼭 필요한 분야인가?
✓
언제 작성된 자료인지 날짜가 나와 있는가?
💬 예를 들어 2010년 자료에 "10년 뒤 드론이 10배 많아질 것"이라는 예측이 있다면? → 이미 그 시점이 지났기 때문에 현재 상황과 다를 수 있어요!
3
🔒 신뢰할 수 있는 정보인가?
찾은 정보를 믿을 수 있는지를 따져 보는 기준이에요. 전문가의 의견이나 과학적 사실에 바탕을 두었는지 살펴봐야 해요.
✓
게시자가 관련 분야의 전문가인가?
✓
출처가 분명한 내용인가?
✓
근거 자료가 뒷받침되어 있는가?
✓
정보를 제공하는 목적이 무엇인가? (교육, 홍보, 판매)
💬 예를 들어 블로그 글에 "드론연구소에서 자격증을 따세요"라는 내용이 있다면? → 홍보·광고 목적일 수 있어서 신뢰하기 어려워요!

❌ 드론 조종사를 소개하는 척 하지만 실제로는 학원 홍보가 목적이에요. 객관적인 직업 정보를 얻기 어려워요.

사전 검색 결과
